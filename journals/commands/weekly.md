
# Your Task

- journals/weekly ディレクトリにある$ARGUMENTSファイルを探す
- $ARGUMENTSに書かれているDaily Noteの各ファイルを探し読み込む ultrathink
- Daily Noteの内容からWeekly NoteのSummaryを作成し、$ARGUMENTSに記入

## Summary

- この週に行なったプロジェクトやタスクについて箇条書きでリストアップ
- 人物に関する評価がメモされている場合、その人物に対する評価をまとめる
- 技術的な課題・検討事項がある場合は「技術的な課題・検討事項」セクションとしてまとめる
- 1on1やミーティングの記録がある場合は、重要な決定事項や話し合った内容を含める
- 新しく導入したツールや環境改善があれば記載する
- プロセスや体制の変更があれば記載する
- 未解決の課題や次週に持ち越すタスクがあれば「持ち越し事項」として記載する
- プライベートな活動（趣味、健康、家族、友人との時間など）があれば「プライベート」セクションとして記載する
- 体調の変化や気づき、個人的な学びがあれば含める
- 適宜日報へのリンクも貼り、情報元がわかるようにする
